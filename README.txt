What: Weather & News Dashboard

Overview: my web application gets user input for a specific city 
and then outputs weather information and top news headlines for that 
location all in one place. 

APIs Used: 
-OpenWeatherMap API: I used this to get current weather data and forecasts.
-NewsAPI: I used this to get access to a large database of news articles wordlwide.

Weather Information Features:
-Displays current weather conditions (temperature, humidity, wind speed) for the specified location.
News Information Features:
-Fetches top news headlines related to the specified location that was inputted.
